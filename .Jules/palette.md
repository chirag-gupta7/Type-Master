## 2025-05-14 - [Skip-to-content Link for Accessibility]
**Learning:** Accessibility standard for this app requires a 'Skip to content' link targeting `id="main-content"` on the main element, visually hidden until focused using Tailwind's `sr-only` and `focus:not-sr-only`.
**Action:** Always ensure root layouts in this design system include this link as the first focusable element.
